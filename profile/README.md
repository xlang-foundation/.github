# The XLang Foundation

The XLang Foundation supports the development of **XLang™**, an open-source programming language for AI, systems, and distributed computing.

## What is XLang?

**XLang™ is an agent-native programming language designed for software that connects models, tools, devices, processes, and distributed nodes.**

It combines Python-like syntax with a native high-performance runtime, and is built for AI, edge, IoT, and system integration workloads.

## Main Repositories

- [xlang](https://github.com/xlang-foundation/xlang) — core language and runtime
- [xMind](https://github.com/xlang-foundation/xMind) — AI memory, planning, and dataflow framework
